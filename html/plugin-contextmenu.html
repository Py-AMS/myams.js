MyAMS "context menu" plug-in

MyAMS "context menu" plug-in is based on standard Bootstrap dropdown menu; it allows to assign a contextual menu, available via a right mouse button click, to any element.

Creating a context menu is very simple:

	<div class="context-menu"
	   data-ams-contextmenu-selector="#myDropdownMenu">
	  Element with context menu
	</div>
	<div class="dropdown-menu" id="myDropdownMenu">
	  <a class="dropdown-item" href="#">First menu option</a>
	  <a class="dropdown-item" href="#">Second menu option</a>
	  <div class="dropdown-divider"></div>
	  <a class="dropdown-item" href="#">Third menu option</a>
	</div>

Menu options are handled as usual, see Navigation to get more information.

MyAMS contextMenu plug-in options

This plug-in options are available as data attributes prefixed by data-ams-contextmenu- or data-ams-menu-:

Name Default Description
selector null JQuery selector of associated dropdown menu

Please note that context menu position should be automatic, based on click event position and browser window size.

You can also get access to native browser's context menu by pressing the <ctrl> key while clicking on the right mouse button.
