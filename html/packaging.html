MyAMS builds

MyAMS is split into several parts:

  • a core, which contains only a very small set of basic features.

  • several modules, which can be integrated into MyAMS package, or downloaded dynamically according to which MyAMS package was used.

  • several plug-ins; these plug-ins are JQuery or Bootstrap plug-ins which have been selected to handle common use cases, like form validation, HTML tables, date and time inputs; they are not included into any MyAMS package, and always downloaded dynamically when needed.

MyAMS is available if three different packaging modes:

  • Default build: this is the default implementation of MyAMS; it's a big bundle (1.2 Mb) which is including JQuery, Bootstrap, FontAwesome icons (in CSS mode) and MyAMS in a single Javascript file built using Webpack; it's this bundle which is currently used by this demo site.

  • Mini-build: this package is only including all MyAMS features in a single bundle; all external dependencies (JQuery, Bootstrap and FontAwesome) have to be included separately. This package have to be used if you want to use FontAwesome SVG icons.

  • Core-build: this really minimalist package only includes MyAMS "core" features; all extensions modules will be downloaded afterwards dynamically, if required.

Please note that all these packages are available in "development" mode as well as "production" mode, where resources are minimized.

MyAMS resources directories are listed below; only files which may have to be included statically into your HTML page are listed, other files exist which are mainly downloaded dynamically:

  • /dist
    • /css
      • /dev
        • myams.css (MyAMS.css, development mode)
      • /ext
        • bootstrap.css (Bootstrap 4 CSS)
        • fontawesome.css (FontAwesome CSS)
      • /prod
        • myams.css (MyAMS.css, production mode)
      • /svgs (FontAwesome SVG files)
      • /webfonts (FontAwesome web fonts)
    • /js
      • /dev
        • myams-dev.js (MyAMS.js, full bundle, development mode)
        • myams-mini-dev.js (MyAMS.js, mini-bundle, development mode)
        • myams-core-dev.js (MyAMS.js, core-bundle, development mode)
        • mod-*-dev.js (MyAMS modules, development mode)
      • /ext
        • jquery.js (JQuery)
        • jsrender.js (Javascript rendering library)
        • bootstrap.js (Bootstrap 4 JS)
        • fontawesome.js (FontAwesome JS)
      • /prod
        • myams.js (MyAMS.js, full bundle, production mode)
        • myams-mini.js (MyAMS.js, mini-bundle, production mode)
        • myams-core.js (MyAMS.js, core-bundle, production mode)
        • mod-*.js (MyAMS modules, production mode)

External resources other than myams.js are required only when using MyAMS packages other then the default build; otherwise, all Javascript and CSS resources are included into this single file, or downloaded automatically when required.
Using Mini or Core builds allows you to change JQuery or Bootstrap packages, or to use FontAwesome in "SVG mode".

You can try other modes using the following links:

  • MyAMS full package

    This bundle includes all required JS and CSS resources.

    	<script type="text/javascript" src="/pkg/js/prod/myams.js"></script>
  • MyAMS mini-package

    This package includes all MyAMS features and modules, but relies on a separate download of JQuery, Bootstrap and FontAwesome; MyAMS CSS file contains all Bootstrap CSS files.

    Using this package allows you to use custom builds of JQuery or Bootstrap; if using a custom Bootstrap build, don't forget to include Popper.js in your dependencies.

    	<script type="text/javascript" src="/pkg/js/ext/jquery.min.js"></script>
	<script type="text/javascript" src="/pkg/js/ext/bootstrap.min.js"></script>
	<script type="text/javascript" src="/pkg/js/ext/fontawesome.min.js"
		data-auto-replace-svg="nest"></script>
	<script type="text/javascript" src="/pkg/js/prod/myams-mini.js"></script>
	<link rel="stylesheet" type="text/css" href="/pkg/css/prod/myams.css">
  • MyAMS core package

    This package includes only MyAMS "core" features, but no additional module; these modules can be downloaded dynamically, according to your own requirements. As for the "mini" package, MyAMS CSS file contains all Bootstrap CSS files.

    	<script type="text/javascript" src="/pkg/js/ext/jquery.min.js"></script>
	<script type="text/javascript" src="/pkg/js/ext/jsrender.min.js"></script>
	<script type="text/javascript" src="/pkg/js/ext/bootstrap.min.js"></script>
	<script type="text/javascript" src="/pkg/js/ext/fontawesome.min.js"
		data-auto-replace-svg="nest"></script>
	<script type="text/javascript" src="/pkg/js/prod/myams-core.js"></script>
	<link rel="stylesheet" type="text/css" href="/pkg/css/prod/myams.css">
