MyAMS "select2" plug-in

Select2 is a common extension to standard HTML "select" element; it provides many features, like filtering and support for AJAX requests, and can be customized in many ways. Look at Select2 website to know all it's available options.

MyAMS doesn't add any custom properties to default Select2 options, but is initializing a default Bootstrap theme, and Select2 language based on current page language; MyAMS also add support for sortable selections, using JQuery-UI sortable plug-in.

You can also use data attributes to define all Select2 callback options, using ams-select2- or just ams- prefix, like in data-ams-select2-matcher="MyApp.matcher", or data-ams-ajax-url="MyApp.urlGetter".

The most basic way to use Select2 is just to add a select2 class on a <select> element:

Basic select form 
	<select id="select_1" class="form-control select2">
	  <option value="1">Option 1</option>
	  <option value="2">Option 2</option>
	  <option value="3">Option 3</option>
	</select>

Multiple select

Another common use of select inputs is to handle multiple selections:

Basic select form 
	<select id="select_1" class="form-control select2" multiple>
	  <option value="1">Option 1</option>
	  <option value="2">Option 2</option>
	  <option value="3">Option 3</option>
	</select>

Multiple orderable select

You can handle selection sorting by just adding a sortable class to the select element:

Basic select form 
	<select id="select_1" class="form-control select2 sortable" multiple>
	  <option value="1">Option 1</option>
	  <option value="2">Option 2</option>
	  <option value="3">Option 3</option>
	</select>

Clearing selection and setting placeholder

You can add a "clear" button by adding a data-allow-clear attribute with a true value, for both single and multiple selections; this can be combined with a data-placeholder attribute:

Basic select form 
	<select id="select_1" class="form-control select2"
	        data-allow-clear="true"
	        data-placeholder="Please select a value">
	  <option value="1">Option 1</option>
	  <option value="2">Option 2</option>
	  <option value="3">Option 3</option>
	</select>
Info

On single select, an empty option is required as first option to prevent selection of the first non-empty option, and allow display of placeholder!

Managing AJAX query

Select2 items can be provided with an AJAX call, using data-ajax attribute to define the API endpoint:

Basic select form 
	<select id="select_1" class="form-control select2"
	        data-ajax--url="data/select2.json"
	        data-minimum-input-length="2">
	  <option value="2" selected>Option 2</option>
	</select>
Info

Please note that when using AJAX request for autocompletion, currently selected options have to be inserted into select as classic options!

MyAMS select2 plug-in options

These plug-in options are available as data attributes prefixed by data-ams-select2-, or just data-ams-:

Name Default Description
theme bootstrap4 Select2 theme name; you must provide your own CSS if using an alternative theme
language page language Current language used to display translated strings
escapeMarkup See Select2 options
matcher See Select2 options
sorter See Select2 options
templateResult See Select2 options
templateSelection See Select2 options
tokenizer See Select2 options
ajaxUrl See Select2 AJAX options. Setting an AJAX URL is also automatically setting minimum input length to 1, except if a greater value was given.
ajaxData See Select2 AJAX options
ajaxProcessResults See Select2 AJAX options
ajaxTransport See Select2 AJAX options
