MyAMS "i18n" module

The i18n module is used to handle internationalization. This is required if you want to display messages in original user's language.

It's up to the web server to include language code into HTML tag's lang attribute; according to that, MyAMS will try to load messages files matching given language, if available.

I18n module API is used automatically by MyAMS on page load; but using strings constants names may return the translated meessage automatically, like in MyAMS.i18n.ERROR_OCCURED.

If you want to provide your own translation file, just create a file called myams-{lang}.js in MyAMS's i18n subfolder; using given french translation file as template is a good starting point...
