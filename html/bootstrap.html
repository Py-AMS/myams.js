Bootstrap addings

MyAMS is adding a few elements to Bootrap-based features.

New CSS classes

  • width-*: unlike Bootstrap's w-* CSS classes, the width-* CSS classes can be used with media queries; for example, you can set a class to width-100 width-sm-75 width-md-50 if you want the element to be 100% width of it's parent by default, 75% on small devices and 50% on medium or larger devices.

  • hidden-mobile, hidden-desktop, visible-mobile-* and visible-desktop-*: these CSS classes are using desktop-detected and mobile-detected classes which MyAMS is applying on HTML body according to detected device type, which is based on user agent and not on screen resolution!

    These classes can be used with media queries, like in hidden-mobile visible-mobile-block-md if you want to hide an element on mobile devices except on medium or larger devices where it will be displayed as block.

New data attributes

  • data-ams-click-dismiss: generally speaking, a click inside a dropdown menu wil dismiss the dropdown; by setting this boolean attribute to false, you can prevent this behaviour, for example if you want to display a form or tabs inside your dropdown (see search box or notifications to see an example).
