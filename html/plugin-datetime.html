MyAMS "datetime" plug-in

MyAMS "datetime" widgets relies on Bootstrap 4 "Tempus Dominus" plug-in (see here for source and documentation).

Empty date and time pickers
Pre-filled date and time pickers

As you can see if looking at this page source, you can associate an « ISO » input to a visible datetime input; this input, if defined using data-ams-iso-target, will be updated automatically with the ISO version of the selected datetime.

MyAMS datetime plug-in options

This plug-in options are available as data attributes prefixed by data-ams-datetime-, or just data-ams-:

Name Default Description
language MyAMS.i18n.language Used to define plug-in and moments.js locale strings
format null Default datetime formatting string
iso-target null If defined, may be a JQuery selector of another input where the ISO version of the entered datetime will be stored.
