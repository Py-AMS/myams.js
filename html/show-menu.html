Show/hide menu action

This action button is displayed at the top right of the main window.

It allows to hide or show the main navigation menu, by setting the hidden-menu CSS class on the HTML body:

	<div id="hide-menu" class="ml-2">
	  <a href="#" class="btn btn-light pt-2"
	     data-ams-click-handler="MyAMS.nav.switchMenu">
	    <i class="fa fa-bars hint" title="Show/hide navigation menus"
	       data-placement="bottom"></i>
	  </a>
	</div>

When the navigation menus are hidden, you can show them on desktop just by moving the mouse cursor to the left of the main window; on mobile devices, you can swipe the screen to the left or to the right to hide or show navigation menus.
