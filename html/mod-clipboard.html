MyAMS "clipboard" module

MyAMS "clipboard" module is used to copy text content to system's clipboard.

MyAMS.clipboard.copy(text)

This function is used to copy given text to system's clipboard.

Note that this function can be selected as a "link target" (see links and navigation), so that the "link" text will automatically be used as source text to be copied into browser's clipboard.
