MyAMS admin layout

MyAMS provides components and stylesheets which have been created with a specific HTML layout in mind, which is the one used by this test site.

Except for a few of them, many of these components are optional, and can be included or not according to your own application needs.

To see how these components are organized into the HTML page, just have a look at the source code; menus entries dedicated to each part are available in the navigation menus.

MyAMS admin layout is globally organized as is:
