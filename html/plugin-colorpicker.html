MyAMS "colorpicker" plug-in

MyAMS "color" widgets relies on Bootstrap 4 "Colorpicker" plug-in (see here for source and documentation).

Empty color picker
Color picker with value and hash prefix
Color picker with custom format and transparency

MyAMS colorpicker plug-in options

This plug-in options are available as data attributes prefixed by data-ams-colorpicker-, or just data-ams-:

Name Default Description
format hex Color format
useHashPrefix false Prefix hexadecimal color value with #
useAlpha false Use transparency in color value
change null Change event listener
pickerChange null Colorpicker change event listener
pickerCreate null Colorpicker create event listener
pickerDestroy null Colorpicker destroy event listener
pickerEnable null Colorpicker enable event listener
pickerDisable null Colorpicker disable event listener
pickerHide null Colorpicker hide event listener
pickerInvalid null Colorpicker invalid event listener
pickerShow null Colorpicker show event listener
pickerUpdate null Colorpicker update event listener
mousemove null Colorpicker mouse move event listener
