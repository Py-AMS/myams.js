MyAMS additions to JQuery

MyAMS is adding a few methods to standard JQuery selectors.

  • $.fn.exists(): this function is checking a JQuery object contents, and returns true if at least one object is contained.

  • $.fn.objectOrParentWithClass(klass): this function is returning the JQuery object itself if it supports the given CSS class, or it's first parent which is supporting it otherwise.

  • $.fn.listattr(attr): this function returns an array containing the value of the given attribute for all objects of the initial JQuery array. This is particularty useful, for example, to get the values of all selected items in a multi-select input:

    const values = $('option:selected', select).listattr('value');

  • $.fn.style(styleName, value, priority): this function can be used to define the style property of the given DOM object, but instead of using the classic JQuery's .css() function, it allows to define the priority of the given style.

    Given arguments must follow the CSSStyleDeclaration (see MDN):

    • styleName: name of the CSS style
    • value: style value to set; if no value is provided, the function returns the value of the given style attribute, or the current global style object if styleName is not defined.
    • priority: set to 1 if the style is "important"

    $('.myStyle').style('border', '1px solid blue', 1);

  • $.fn.removeClassPrefix(prefix): this function acts as $().removeClass, but removes all CSS classes names which starts with given prefix.

    $('i', container).removeClassPrefix('fa');
