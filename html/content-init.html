Initializing contents

MyAMS allows you to download or generate HTML content dynamically; when such a content contains elements which may be handled via plug-ins, they have to be initialized as well!

The MyAMS.core.initContent(element) function (which can be overriden using data attributes or by changing MyAMS default configuration, see core module description) is used to intialize any HTML content included dynamically into the DOM; this function will call the initElement(element) function of all MyAMS modules which have been loaded, for example to handle any form or skin element included into these parts, before calling MyAMS.registry.run(element) to be able to initialize all registered plug-ins into the new content.
