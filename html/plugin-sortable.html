MyAMS "sortable" plug-ins

MyAMS sortable features are also based on JQuery UI package. MyAMS is just a thin wrapper around JQuery UI sortable plug-in, and his documentation is the best source of information to know how it work.

A simple sortable element can be created like this:

	<div class="parent clearfix">
	  <div class="sortable">
	    <div class="border mouse-move ml-3 mb-3 width-200px height-100px text-center">1</div>
	    <div class="border mouse-move ml-3 mb-3 width-200px height-100px text-center">2</div>
	    <div class="border mouse-move ml-3 mb-3 width-200px height-100px text-center">3</div>
	    <div class="border mouse-move ml-3 mb-3 width-200px height-100px text-center">4</div>
	    <div class="border mouse-move ml-3 mb-3 width-200px height-100px text-center">5</div>
	  </div>
	</div>

MyAMS sortable plug-in options

These plug-in options are available as data attributes prefixed by data-ams-sortable-:

Name Default Description
connectWith false selector of other sortable elements that the items from this list should be connected to
containment false constrains dragging to the bounds of the element specified by this option; can be a JQuery selector, a DOM element or a custom string ("parent", "document" or "window")
handle false if specified, restricts sort start click to the specified element
helper "original" allows for a helper element to be used when dragging; can be set to clone to drag a copy of the original element, or to a function which will return a DOM element used for dragging
items "> *" specifies which elements inside the sortable should be sortable
options undefined serialized JSON object containing additional sortable settings; see JQuery-UI sortable documentation
over undefined name of an event handler triggered when a sortable item is moved over a sortable list
placeholder false class name that is applied to the otherwise white space during dragging
start undefined name of an event handler triggered when sorting starts
stop undefined name of an event handler triggered when sorting stops
