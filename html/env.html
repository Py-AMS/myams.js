MyAMS environment

MyAMS.env is an object used to store current MyAMS environment; this object should not be replaced or changed after page initialization.

Current environment properties are:

  • bundle: boolean value specifying if MyAMS was loaded from global bundle or from a mini or core package.

  • devmode: boolean value specifying is MyAMS was loaded in development mode.

  • devext: string used to define name of MyAMS resources; this value is -dev when MyAMS is in development mode, or an empty string otherwise.

  • extext: string used to define name of external resources, like JQuery and Bootstrap; this value is .min when MyAMS is in development mode, or an empty string otherwise.

  • baseURL: base URL of MyAMS resources; this is the name of the directory (with trailing "/") from which MyAMS.js was downloaded.
