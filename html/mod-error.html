MyAMS "error" module

This module is used to display errors messages as alerts, for example in a form.

MyAMS.error.showErrors(parent, errors): Promise<>

This function is used to add an alert into given parent element.

As this function relies on other modules, she returns a Promise which is resolved when all missing modules have been checked and alerts included into DOM.

If errors is a simple string or an array of strings, there strings are simply added to the alert (see alert module).

If errors is an object, it can provide more informations, including specific messages for selected input widgets; see Forms errors to know the format of JSON errors.

MyAMS.error.showHTTPError(error): Promise<>

This function is used to display a message box when an HTTP error is returned as a JSON object, containing at least code, title and message attributes.

As the function relies on other modules, she returns a Promise which is resolved when all modules have been loaded.
