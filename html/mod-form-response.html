Managing server response

When submitting a form, several typical responses types can be received:

  • JSON response: the server send a JSON response containing properties defining how the response should be treated. This JSON response is handled by MyAMS AJAX module, including errors management.

  • HTML response: the server send an HTML response which is directly placed into a target element ; see form options to know all form settings.

  • text response: as for HTML responses, the received text is placed into a target element.

  • file response: the server send a file object which may be downloaded and saved by the browser. This process generally requires the creation of a dedicated iframe, which is done automatically by MyAMS as soon as a few conditions are required; this configuration is described into Managing file downloads section.
