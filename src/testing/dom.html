Main DOM objects

MyAMS.dom is a simple object which is storing JQuery references to several main DOM objects of the HTML page. These are:

  • page: pointer to html container.

  • root: pointer to body container.

  • nav: pointer to main navigation bar (nav tag); go to navigation menus to know how to initialize navigation bar.

  • main: pointer to main panel (#main).

  • leftPanel: pointer to left panel container (#left-panel).

  • shortcuts: pointer to shortcuts panel (#shortcuts).
