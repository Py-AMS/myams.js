Body classes

MyAMS supports several custom CSS classes which can be applied to the main <body> tag:

  • minified: the main navigation menu can be displayed in a classic way or in a "minified" way, in which only first level icons are visibles; sub-menus are then visible on mouse-over.

    A "minifying" button is present, by default, below the navigation menus.

  • hidden-menu: in this mode, the navigation menu is nearly hidden and displayed on mouse over the left part of the screen.

    A "show/hide menu" button is present, by default, into page's header, to handle this class.

    Notice

    Please note also that the minified class should not be used with the hidden-menu class at the same time!

  • fixed-ribbon: when this class is applied, the ribbon is fixed instead of scrolling with the #content part.

  • desktop-detected and mobile-detected: these CSS classes are assigned automatically after page load according to detected device type.

minified and hidden-menu classes are stored into browser's local storage and restored automatically on startup.
