MyAMS loading and base configuration

Whatever packaging you use, including MyAMS.js into your resources will create a global MyAMS object.

This object has several properties:

  • $: pointer to JQuery main function.
  • env: current MyAMS environment.
  • config: base MyAMS configuration.
  • dom: commonly used DOM objects.
  • core: MyAMS core features and functions.
  • require: MyAMS function used to check and download additional modules.
  • registry: MyAMS plug-ins registry
