Modal forms

Modal forms are nothing but classic forms, but which are displayed into a modal dialog.

You can see an example of a modal form here.

Everything is handled as in a classic modal and in a classic form, except for the following points:

  • the modal is closed automatically after submit, except if errors occured or if an attribute into JSON response is set for the modal to be kept open.

  • you can assign a target on a modal form as well; this target will be updated with submit's response content, as usual. See this example, just submit the form to see how it works!
