Basic forms

This is the simpliest possible form that MyAMS can handle.

Compared with basic Bootstrap form template, you can notice a few addings:

  • hints are added after labels.

  • a required class can be added to labels attached to required fields; if so, labels are displayed (by default) with bold fonts and hints markers are displayed in red color.

  • a text-sm-right is added to labels, so that labels are aligned on the right on wide enough displays.

  • input fields are wrapped inside a div class="form-widget" element; it's inside this block that errors messages attached to an input field, if any, are inserted.
